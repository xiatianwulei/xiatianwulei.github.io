📘 Swift 高级函数速查手册
Swift 中常用的高阶函数包括：map / filter / reduce / compactMap / flatMap / sorted / contains / first / firstIndex 等。本手册提供最简明示例，便于开发中快速查阅。

1. map — 映射（对每个元素做转换）

    ```
    let numbers = [1, 2, 3]
    let strings = numbers.map { "数字\($0)" }
    // ["数字1", "数字2", "数字3"]
    ```

2. filter — 过滤（筛选元素）
    ```
    let numbers = [1, 2, 3, 4, 5]
    let evens = numbers.filter { $0 % 2 == 0 }
    // [2, 4]
    ```
3. reduce — 汇总（将数组合成为一个值）
    ```
    let numbers = [1, 2, 3]
    let sum = numbers.reduce(0) { $0 + $1 }
    // 6
    ```    
4. compactMap — 映射并自动移除 nil
    ```
    let inputs = ["1", "2", "a", "3"]
    let numbers = inputs.compactMap { Int($0) }
    // [1, 2, 3]
    ```    
5. flatMap — 扁平化（展开多维数组）
    ```
    let arr = [[1, 2], [3, 4]]
    let flat = arr.flatMap { $0 }
    // [1, 2, 3, 4]
    ```
6. sorted — 排序
    ```
    let arr = [3, 1, 2]
    let sortedArr = arr.sorted { $0 < $1 }
    // [1, 2, 3]
    ```
7. contains(where:) — 判断是否存在满足条件的元素
    ```
    let arr = [1, 3, 5]
    let hasEven = arr.contains { $0 % 2 == 0 }
    // false
    ```

10. 取出数组里面的元素组成一个新的数组

    ```
    let names = entity?.gradeEquityList?.compactMap { $0.gradeName } ?? []  
    ```
    
11. 取出数组里面的元素组成一个新的数组

    ```
    let totalCount: Int = filteredList.reduce(into: 0) { (result, item) in
    // 闭包参数：
    // - result：累加的中间结果（初始值是 0，每次循环会更新）
    // - item：当前遍历到的 filteredList 中的元素（PrizeVo 类型）
    let amount = item.amount ?? 0 // 如果 amount 是 nil，就用 0 代替
    result = result + amount // 把当前 item 的 amount 加到 result 中
    }
    ```
    
12. 取出数组里面的元素组成一个新的数组

    ```
    let filteredList: [PrizeVo] = prizeVoList.filter { (item) -> Bool in
    // 闭包参数：item 是数组中的每个元素（PrizeVo 类型）
    // 逻辑：判断 item 的 prizeType 是否为 11（如果为 nil 则默认 0）
    let type = item.prizeType ?? 0 // 如果 prizeType 是 nil，就用 0 代替
    return type == 11 // 返回 Bool 结果，true 则保留该元素
    }
    ```
